Reveal.js

Reveal.js is an open-source HTML framework allowing to create beautiful presentations supporting

  • authoring via MarkDown,
  • math equations with MathJax,
  • syntax highlighting for code, and
  • much more.

This presentation showcases examples created with the reveal.js plugin collection.
